The idea behind my first project is to showcase my skill at writing algorithms. I have written 3 algorithms for this project. Try them out for yourself! Here's what each does:

  • Rot13 encrypts your message using the ROT13 method.
  • Seconds allows the user to enter any amount of time in seconds and get back the time in human readable format.
  • Fibonacci checks whether a number is the product of 2 consecutive Fibonacci numbers.
